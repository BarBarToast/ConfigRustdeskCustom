# ConfigRustdeskCustom

Repository untuk menyimpan config custom RustDesk.

## Cara Import Config

1. Open RustDesk
2. Settings → Network
3. Buka menu ID/Relay Server
5. Copy isi config di aturwalpat-config.txt
6. Paste lalu Apply

## Notes

- Repository ini hanya menyimpan config import
- Jangan upload private key server
